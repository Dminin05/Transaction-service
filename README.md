### 1. Clone the repository
```sh
   git clone https://github.com/Dminin05/Transaction-service.git
```
### 2. Build project
```sh
   ./gradlew build
```
### 3. Docker compose
```sh
   docker compose up
```

## Here you can see swagger documentation: 
```sh
   http://localhost:8080/swagger-ui/index.html
```
